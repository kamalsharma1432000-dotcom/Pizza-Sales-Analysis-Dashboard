This project showcases an end-to-end Data Analytics solution using Microsoft Power BI, focused on analyzing pizza sales data to extract meaningful business insights.

The dashboard helps stakeholders understand sales performance, customer behavior, and product trends for better decision-making.

⸻

🎯 Objectives

* Evaluate overall sales performance and revenue trends
* Analyze customer ordering patterns
* Identify top-performing products and categories
* Monitor time-based sales trends (daily/monthly)
* Deliver actionable insights using interactive dashboards

⸻

📊 Key Metrics (KPIs)

* Total Revenue
* Total Orders
* Total Quantity Sold
* Average Order Value (AOV)
* Top-Selling Pizza Categories
* Best Performing Products
* Sales Trends (Daily & Monthly)

⸻

🛠️ Tools & Technologies

* Microsoft Power BI
* Power Query
* DAX
* Excel / CSV Dataset
* Data Analysis Techniques

⸻

📂 Dataset Description

The dataset contains:

* Order ID
* Order Date
* Pizza Name
* Category
* Size
* Quantity
* Unit Price
* Total Sales

Data was cleaned and transformed using Power Query.
